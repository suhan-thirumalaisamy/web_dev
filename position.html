INSET

position:static;

An element with position:static ; is not positioned in any special way; it is always positioned according to the normal flow of page;

This div element has position as static.

position:relative;

An element with position:relative ; is positioned relative to its normal position;

This div element has position as relative.

position:fixed;

An element with position:fixed; is positioned relative to the viewport which means it always stays in same place even if page is scrolled;

This div element has position as fixed.

position:absolute;

An element with position:absolute ; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed);

This div element has position as relative.
This div element has position as absolute.
